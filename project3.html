Visual

The site uses parallax scrolling to animate elements of the page as the user scrolls, creating several cinematic scenes which are pleasing to the eye. The site effectively demonstrates the vibrant scenes from the movie.

Technical

The site gives off a very complex feel, which highlights the amount of work dedicated into producing the movie. For a project with such a large budget spent on CG, the accompanying site must be just as graphically impressive.

Immersive

Linear top-to-bottom flow of the site leads the user through a story about each part of the project. This engrossing site structure almost guarantees that the user will scroll down to the bottom of the site, by which time they have been sold on the movie.
